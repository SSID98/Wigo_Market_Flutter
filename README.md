# 🛒 E-Commerce Mobile App (Flutter)

A fully functional e-commerce mobile application built using Flutter, focused on delivering a smooth user experience and seamless API integration.

---

## 📱 Overview

This application allows users to browse products, view details, and perform core e-commerce actions through a clean and responsive mobile interface.

I worked as the **Frontend (Mobile) Developer**, responsible for building the UI and integrating backend APIs provided via Swagger.

---

## 🚀 Features

- 🛍 Browse products
- 🔍 View product details
- 🧑 User profile management
- 🛒 Add to cart functionality
- 💳 Payment flow integration (via API)
- 🔗 API integration using REST endpoints
- ⚡ Smooth and responsive UI

---

## 🛠 Tech Stack

- **Flutter (Dart)**
- REST APIs (Swagger-based)
- Git & GitHub (Version Control)

---

## 👨‍💻 My Role

- Developed the complete mobile UI using Flutter
- Integrated backend APIs provided by the backend developer
- Handled state updates and data rendering
- Collaborated with backend team to ensure proper API communication
- Implemented clean and maintainable code structure

---

## 🔗 API Integration

The app consumes REST APIs provided via Swagger documentation.  
Responsibilities included:

- Connecting endpoints to UI
- Handling API responses
- Managing loading and error states
